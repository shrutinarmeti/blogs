# Shruti Narmeti — Blog

A static finance blog hosted alongside [shrutinarmeti.github.io](https://shrutinarmeti.github.io).

## Structure

```
index.html          # Blog listing page
styles.css          # Shared stylesheet
posts/
  the-power-of-compound-interest.html
  index-funds-vs-active-management.html
  building-an-emergency-fund.html
```

## Adding a New Post

1. Create a new HTML file in `posts/` using an existing post as a template.
2. Add a new `<article class="blog-card">` entry in `index.html`.

## Theme

Dark purple developer portfolio theme, consistent with [shrutinarmeti.github.io](https://shrutinarmeti.github.io). Every page includes a **Back to Portfolio** button in the header.
